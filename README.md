# H5P Sharing Custom
This WordPress plugin is a fork of [H5P Sharing](https://github.com/otacke/h5p-sharing) plugin.

## Features
* QR Code now link direct to the link of the content and not at a html tag
* When viewing H5P content in the editor, QR Code is downloadable as SVG.

## Install/Usage
Use github's download option to download this archive as a ZIP file and install _H5P Sharing_ by uploading the plugin archive in your WordPress backend
and activate the plugin. Done.
